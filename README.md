# mapper

Ejercicio tipo quiz

## Para correr el proyecto

- Tener [node](https://nodejs.org/en/)
- Clonar repo
- Ejecutar `npm install` para instalar dependecias (ts-node, typescript, etc...)
- Ejeutar `npm start` para correr el proyecto
